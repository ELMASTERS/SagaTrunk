A introduction to Saga...

Saga is a open source server project for Ragnarok Online 2. Currently the were developing it under closed source until the game is officially released. Saga Revised is a side branch of the current saga project which has been developed with the purpose to make it more user-friendly. In order to make it more user-friendly, most of the scripting as seen in the main saga project has been replaced by tabular data with some additional xml files to configure the behavior.

Both projects are written mainly in c-sharp (c#) and are released under the CCPL (Creative Commons Public License). You may download the source code from svn at the address https://svn1.hosted-projects.com/SagaE/SagaRev/. To read more about subversion please refer to: http://en.wikipedia.org/wiki/Subversion_(software)

To addition to cross-platform we support mono. Mono is a open source framework similar to Microsofts .net framework. As mono is still in development itself we cannot guarantee that we completely support Linux but with companies such as Novell supporting the mono framework we are confident in its success.
