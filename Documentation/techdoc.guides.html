Fileformat: Guides

The warper fileformat is used to configure the Npcfunction 'GuideConversation'. Used by the npc's which provide a locationguide interaction.

<?xml version="1.0" encoding="utf-8" ?>
<guide menu="6">
  <point type="npc">1006</point>
  <point type="npc">1011</point>
  <point type="npc">1010</point>
  <point type="npc">1013</point>
  <point type="npc">1007</point>
  <point type="npc">1008</point>
  <point type="npc">1009</point>
  <point type="npc">1012</point>
  <point type="npc">1014</point>
  <point type="npc">1050</point>
  <point type="npc">1051</point>
  <point type="npc">1055</point>
  <point type="npc">1056</point>
  <point type="map">1</point> <!-- Northwest of Prontera //-->
  <point type="map">1</point> <!-- Southwest of Prontera //-->
</guide>

In the example above you see the default fileformat used for our locationguides. This is a new function and still expirimental please wait on futher documentation prior on using this.
