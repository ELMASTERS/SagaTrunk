How are the password stored?

The password is stored in as an MD5 Hash. The game itself sends a modified MD5 Hash so to make it easier for all the CP developers out there we've issued our database plugin to handle this.

I want to know modify my client, where to start?

We don't provide any support for modifying the client neither will we in the nearby future. Ask somewhere else.

I found a bug where to i submit it?

To submit bugs you can create a account at our bugtracker: http://onaasystems.net/mantis/. Make sure your bug isn't double posted.

Can i join your development staff?

You have to require a certain profile. We aim on developers who have sufficient expierence in c-sharp. If you fit that profile we're more than happy to have a talk with you.

When i login i get some korean error?

The crc has a mismatch. Simply said this means your client version isn't compatible with the server version. Consider upgrading or downgrading.

How can i add accounts to my server?

We insist that you have a third party CP to allow self-registrations. We're often develop inhouse software ourselves which we later on make available. If you just need a few test accounts you can start the login server and type: 'account.create username password female|male'.

Where can i get realtime support?

We have a irc channel on deltaanime. Please visit #saga on deltanime for your questions/support, or just for chatting!

Is there PvP and Guilds?

No the official server does not have guilds or pvp yet neither have we. They are will be added in the near future.

What can i customize?

You can alter: AI, Npc templates, Npc Functions, Skills, Quests, Quests storage engine, Database Storage engine, Fileformats, add custom managers.

I don't like MySQL can i use a other database?

Yes you can we provide a complete database structure on the setup page found here. And our MySQl plugin's source code can be found here in the abstraction folder

I'm using Windows and i get a application error at the startup?

Either your configuration file is invalid or you are missing the .net framework. Saga requires a at least .net 2.0 framework you can download this from microsoft using the windows update or from http://www.microsoft.com/

When i start the server it says it's found errors while loading

Once the server has one error it may lead to a chain of errors. Most errors are related to bad fileformat or incorrect setup suggested configuration of directories. Make sure non of the files are missing and configuration is valid otherwise request help on the forum: http://sagaro2.info/phpbb3/"

Does saga run on linux?

To run saga under a linux server you need to use Mono. Which is a alternative framework similair to .net from Microsoft (You can use mono also under windows).

How are quests made?

We have made a sample pluging to script quests on a similiar way it's done on kRO2. It's a lightweight solution that allows plug-and-play based quests.

How are npc's made?

You add a npc to the tabular data for example ~\Data\npc-spawns\hodehimes.csv. Make sure the coordinates are valid and use a existing class for example 'Saga.Templates.Npc' this class is added in the scripting section by default. That's it!
