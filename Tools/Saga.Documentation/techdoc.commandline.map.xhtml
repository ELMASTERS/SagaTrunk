This is the documentation reference for commandline interface for the saga map server. player -rescue Warps an player to the speciafied mapid of choose. player -isonline Checks if the speciafied character is online right now. player -zeny Gives the speciafied player an amount of rufi. (10.000 rufi is 1 zeny) player -kick Kicks the player with the speciafied name. (case-insensetive) server -broadcast Broadcasts a message to all players currently logged in. server -toggledc This will cause the server not to disconnect the player on a uncaught error. An error is occuring in most cases due cheating. All simple errors that the client can display are all caught and show up in the client. maintenance -force -release This will release the maintenance lock on the authentication server for your spefic server. With a maintenance lock you can disallow new people to join your server. maintenance -force -enter This will open the maintenance lock on the authentication server for your spefic server. With a maintenance lock you can disallow new people to join your server. maintenance -schedule This will schedule a reboot of the server. This means susspending the server into an maintenance mode first, kicking all players, wait a time for database processing, exit the process.
